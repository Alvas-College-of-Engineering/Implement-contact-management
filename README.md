# Implement-contact-management
Implement contact management


## 📌 Project Overview

The Contact Management System is a Java-based web application developed to manage contact information efficiently. The application allows users to add, search, and manage contacts using a simple and user-friendly interface.

This project is developed using Java Servlets, JSP, HTML, CSS, and Apache Tomcat.

---

## 🚀 Features

- Add new contacts
- Search contacts
- View contact details
- Simple and responsive user interface
- Servlet-based backend processing

---

## 🛠️ Technologies Used

- Java
- JSP (Java Server Pages)
- Servlets
- HTML
- CSS
- Apache Tomcat Server

## ⚙️ Setup Instructions

### 1. Clone the Repository

```bash
git clone https://github.com/Alvas-College-of-Engineering/Implement-contact-management.git
```

### 2. Import Project

- Open Eclipse Enterprise Edition
- Go to File → Import
- Select Existing Projects into Workspace
- Choose the project folder

### 3. Configure Apache Tomcat

- Download and install Apache Tomcat
- Add Tomcat Server in Eclipse
- Configure the project with the server

### 4. Run the Project

- Right-click the project
- Select Run on Server

Open in browser:

```text
http://localhost:8080/ContactManagementSystem/
```
